# Blaze Doom Adventure 32X

![Titlescreen](https://github.com/gameblabla/BlazeDoomAdventure32X/blob/main/titlescreen.png?raw=true)

The original release had a fatal flaw : data was not properly aligned.

Thus the game failed to run on FPGA core, Picodrive or real hardware.

It only ever ran on Kega Fusion. This is now fixed.

# Description

Traverse through 15 levels, from Sonic Zones to the Mushroom Kingdom!

You play as Blaze the Cat, who is unexpectedly approached by an older Doctor Eggman Nega claiming to be from an alternate future timeline.

In his future, kingdoms have collapsed into hostile wastelands, dimensions bleed into one another, and chaos spreads unchecked.

Nega claims there is only one way to stop this future:

Blaze must travel across fractured worlds and create a time paradox strong enough to destabilize the doomed timeline and restore balance.

Armed with strange weapons provided by Nega—and unsure whether she can truly trust him—Blaze begins her adventure.
